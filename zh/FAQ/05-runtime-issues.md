## 运行相关

### 雷电模拟器

- 关闭自动旋转屏幕：设置 → 其他设置 → 关闭。
- 调整分辨率：设置 → 性能设置 → 推荐「手机版-720×1280-DPI240」

### 连接正常，但是操作卡顿、异常、频繁出错或显示任务已完成

- 异形屏UI适配设为0。
- 主页 → 启动游戏 → 齿轮 → 选择正确游戏客户端版本。
- Adb Input触控模式慢属正常，自动战斗建议换其他模式。
- 繁中适配持续优化中，请保持`MaaYuan`为最新版本。

### 运行任务后无反应或持续点击模拟器某处

- 重启MaaYuan
- 多开冲突：检查模拟器端口与`MaaYuan`显示端口是否一致。
- 后台保活：取消勾选「后台挂机时保活运行」（右上角菜单 → 设置中心 → 其他）。
- 检查分辨率：确保分辨率为720*1280。
- 更换触控方式：在`MaaYuan`设置 → 连接设置 中，选择其他触控方式进行尝试。

### biubiu加速器无法启动

请将此任务放在启动游戏任务之前，此任务只负责加速不负责启动游戏。（如果加速了就启动是因为biubiu有加速自动启动游戏的功能）

### 截图用时过长/触控反应时间过长

- 截图耗时≤1000ms属正常。
- 启用GPU加速推理（`MaaYuan`设置 → 高级设置），核显/老旧独显（如GTX 750Ti以下）不建议开启；遇bug则关闭。
- **检查分辨率**：[新手教学](XXX)。

### 启动游戏无法运行

- 请根据客户端-系统设置-游戏语言，选择`MaaYuan`的资源类型，港服务必确认此项。

## 任务相关

### 6-24/7-15运行失败

- 确认模拟器分辨率为720*1280。
- 未通第六章时，先进入第六章并停在6-24关卡外，再开始任务。
- **MuMu模拟器**：取消勾选「后台挂机时保活运行」（右上角菜单 → 设置中心 → 其他）。
- DPI≥180（过低会导致画面模糊、OCR识别错误）。
- `MaaYuan`设置 → 高级设置 → 关闭GPU加速。
- 卡在主线章节滑动？重装`MaaYuan`。

### 每日进膳任务失败

- 更换代号鸢/如鸢游戏背景（UI更新需反馈完整截图等待适配）；
- 调换每日进膳与据点任务顺序。

### 心纸营建卡在宝箱处

OCR识别偶有误差，已优化，仍遇问题请反馈。

## 作业相关

### 如何保证自动战斗的正确运行

- 战斗页面右侧详情/自动/倍速条保持展开。
- 按作者站位摆放密探，确保密探核心命盘一致（尤其+能量）。
- `MaaYuan`无法探查技能使用，需完美对轴。

### 放入作业后提示“资源加载失败”，导致无法连接模拟器

- 作业json需放`resource\copilot-cache`下。
- 推荐神秘代码导入，若仍有问题，换作业尝试，有问题的文件在官方群找管理处理。
- 取消激活或删除导入作业，删除后重启，不行则重装。

### 神秘代码导入失败

网络问题致无法与作业站通讯，可下载json本地导入或稍后重试。

### 打到第七回合自动结束

- 确保未选抄完几回合后自动停止。
- 调整模拟器分辨率至720*1280后尝试。
- 还不行就用战斗中开始抄作业，从第七回合开始。

### 抄作业失败

- 0s 结束为未激活作业，检查作业是否导入并激活，正确激活作业运行后会提示作业版本信息：“作业信息：由 MaaYuan Share vXXX生成，需MaaYuan vXXX 上运行。”，为例行提示，如能正常运行则无需理会；
- 10 - 20s 左右结束为未识别到指定内容（如自动刷洞窟的“前往讨伐”；自动抄作业的“开始战斗”；回合内开始抄作业的右上角“回合X”），检查开始任务的游戏界面是否正确。
- 导入神秘代码后无法连接/提示资源加载失败/抄作业无信息、无法运行，下载最新版本MaaYuan同时确保作业是近期生成的。
