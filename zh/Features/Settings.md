## 外部通知

支持任务完成后发送“任务已全部完成”或自定义内容的通知。

### 钉钉

- 电脑端钉钉建群并添加自定义机器人：群设置 → 机器人 → 添加机器人 → 自定义 → 添加。

- 安全设置选“加签”，复制记录密钥和 webhook 地址。

- 打开`MaaYuan`→ 设置 → 外部通知 → 开启钉钉通知，将密钥和地址分别粘贴至对应位置：

- 机器人 Token = webhook 地址

- 钉钉 Secret = 密钥

- 可选填自定义内容，点击“发送测试”验证。

### 邮箱

- 登录邮箱网页端，进入设置 → 账号与安全 → POP3/IMAP/SMTP/Exchange/CardDAV 服务，开启服务并生成授权码（不同邮箱步骤类似）。
- 复制授权码，粘贴至`MaaYuan`指定位置，点击“发送测试”接收验证通知。
- 机器人 Token = 你的 QQ 邮箱账号（以 QQ 邮箱为例）
- 邮箱 Secret = 生成的授权码
