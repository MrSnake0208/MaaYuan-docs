# 设置界面<Badge type="warning" text="设置" />

## 定时执行

- 可设定执行时间和频率，设定完成后，会在指定时间执行当前配置勾选的任务，需保证`MaaYuan`与其所连接的模拟器为开启状态。
- 内置“强制定时启动”选项，勾选后`MaaYuan`会在指定时间停止当前任务，重启模拟器并开始新任务，需在设置-启动执行中填写“模拟器路径”、“附加命令”及“等待模拟器启动时间”（多开模式下不生效）。

## 性能设置

GPU加速开关，可自行调整。

## 运行设置

主要用于控制任务运行过程及调试辅助功能设置。可根据需要进行基础配置，调试或排障时建议开启截图保存和实时视图功能。

- 任务运行失败后是否继续运行其它任务：建议打开。
- 实时视图及刷新率设置：为节约电脑运存，可选择关闭。

## 连接设置

- 捕获方式及触控方式：任务执行失败/启动任务无反应时，可自行尝试更换。
- 记住连接：建议打开。
- 其余设置可根据个人习惯调整。

## 启动设置

启动设置仅在单开时生效。

- 模拟器路径：选择模拟器.exe文件所在的路径，可选桌面快捷方式，如mumu模拟器为MuMuNxMain.exe。
- 附加命令：多开时需填写，可点击“多开快捷配置”进行选择。
- 等待模拟器启动时间：自行调整。

## 界面设置

可根据个人习惯进行主题色、背景图片设置、应用程序显示语音、界面字体大小缩放、最小化设置、通知开关的设置。

## 外部通知

支持任务完成后发送“任务已全部完成”或自定义内容的通知。

### 钉钉

- 电脑端钉钉建群并添加自定义机器人：群设置 → 机器人 → 添加机器人 → 自定义 → 添加。

- 安全设置选“加签”，复制记录密钥和 webhook 地址。

- 打开`MaaYuan`→ 设置 → 外部通知 → 开启钉钉通知，将密钥和地址分别粘贴至对应位置：

- 机器人 Token = webhook 地址

- 钉钉 Secret = 密钥

- 可选填自定义内容，点击“发送测试”验证。

### 邮箱

- 登录邮箱网页端，进入设置 → 账号与安全 → POP3/IMAP/SMTP/Exchange/CardDAV 服务，开启服务并生成授权码（不同邮箱步骤类似）。
- 复制授权码，粘贴至`MaaYuan`指定位置，点击“发送测试”接收验证通知。
- 机器人 Token = 你的 QQ 邮箱账号（以 QQ 邮箱为例）
- 邮箱 Secret = 生成的授权码
